# Clean Energy Optimizer

Aplikacja frontendowa do optymalizacji ładowania na podstawie miksu energetycznego.

---

## Technologie
- React
- Typescript
- Material UI
- Testy: Vitest + MSW

---

## Funkcjonalności

- **3-dniowy miks energetyczny**
    - Pokazuje miks energetyczny dla 3 kolejnych dni
    - Wyświetla udział poszczególnych żródeł energii w formie wykresu kołowego
- **Obliczanie optymalnego okna ładowania**
    - Formularz do wyboru liczby godzin ładowania
    - Pokazuje datę rozpoczęcia i zakończenia optymalnego ładowania, oraz udział czystej energii w tym czasie
